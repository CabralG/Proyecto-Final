¿Qué hace un Auto Explorador?

Vigila o inspecciona distintos terrenos demandando menos esfuerzo físico y agilizando las actividades.

Digital

El Lenguaje de Programcion Utilizado es Arduino similar a C++

Analogico

Utilizamos una placa Arduino MEGA, junto con un Módulo Wifi y un Módulo ESP32cam

Control

El Auto se controla mediante una App Movil

Auto Explorador

Un fundamento muy importante de este proyecto es la exploración urbana, para no correr algún tipo de peligro al momento de salir al exterior ya que un auto explorador es una herramienta muy útil para la exploración.

Este es nuestro, Proyecto Final

Un Auto Explorador controlado por Arduino via WiFi, Tiene un alcance aproximado de 20 metros, cuenta con 4 baterias de litio, un camara de 2Mpx que transmite en ViVo desde el auto.
